## üì¶ Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd ataussamad-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Replace Profile Image**
   - The current profile section uses an emoji placeholder (üë®‚Äçüíª)
   - To add your photo, replace the emoji div in `app/components/Hero.tsx` with an Image component
   - Recommended size: 300x300px or higher (square aspect ratio)

4. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

5. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## üé® Customization

### Profile Image
Replace the placeholder image in `public/profile-placeholder.jpg` with your actual photo.

### Personal Information
Update the following files with your information:
- `app/components/Hero.tsx` - Name and title
- `app/components/About.tsx` - About section content
- `app/components/Contact.tsx` - Contact information
- `app/components/Projects.tsx` - Your projects
- `app/components/Education.tsx` - Educational background

### Colors and Styling
Modify `tailwind.config.js` to change the color scheme:
```javascript
colors: {
  primary: {
    500: '#667eea', // Change primary color
  },
  secondary: {
    500: '#764ba2', // Change secondary color
  }
}
```

### Projects
Update the projects array in `app/components/Projects.tsx` with your actual projects:
- Add project descriptions
- Update technology stacks
- Add project links
- Change project icons/images
